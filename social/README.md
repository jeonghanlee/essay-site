# Social Announcement Drafts

## Scope

This directory stores human-reviewed announcement drafts for social channels.

**Out of scope:** access tokens, API clients, automatic publishing jobs, scraped engagement data, and platform credentials.

## Policy

Social channels announce new essays. They do not store canonical essay text.

Each draft should contain:

- source essay path
- target channel
- draft status
- final canonical URL
- short post text

Use one file per channel when tone or length differs.
